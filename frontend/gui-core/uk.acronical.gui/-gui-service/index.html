GuiService

open class GuiService

A centralised service for managing GUI interactions and inventory events.

This service utilises the Bukkit org.bukkit.plugin.ServicesManager to facilitate cross-plugin sharing of GUI logic, ensuring that the most recent version of the framework handles inventory events.
