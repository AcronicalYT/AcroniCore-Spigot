MessagesConfig

open class MessagesConfig : DefaultConfig

Handles retrieval of messages from a YAML configuration file, providing default values and logging warnings for missing messages.

This class extends DefaultConfig to provide automatic colour code translation and logging warnings when requested keys are missing from messages.yml.
