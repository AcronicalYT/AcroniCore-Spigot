ChatInputListener

open class ChatInputListener

A utility class for capturing player chat as functional input.

This listener allows developers to "prompt" a player for text input via chat. It intercepts the next message sent by the player, cancels the event to prevent it from appearing in global chat, and executes a provided callback.
