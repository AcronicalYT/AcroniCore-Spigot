TaskManager

open class TaskManager

A central management class for scheduling and managing plugin tasks.

This class acts as a wrapper for the org.bukkit.scheduler.BukkitScheduler, providing a simplified API for synchronous and asynchronous execution, as well as access to specialised utilities like TaskChain and CountdownBuilder.
